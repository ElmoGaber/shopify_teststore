# Shopify Hiring Assessment – EcomExperts

A pixel-perfect implementation of the EcomExperts Shopify Developer Hiring Assessment using Shopify Online Store 2.0.

## Overview

This project recreates the provided Figma design as a dedicated Shopify page using custom-built sections only.

The implementation was developed from scratch without reusing any of Dawn's built-in content sections, following the assessment requirements.

## Features

- Custom Banner section
- Custom Product Grid section
- Theme Customizer support
- Product Picker support
- Dynamic product popup
- Dynamic variant rendering
- AJAX Add To Cart
- Responsive Desktop / Tablet / Mobile
- Accessibility support
- Semantic HTML
- Shopify Online Store 2.0 compatible

## Technologies

- Shopify Liquid
- HTML5
- CSS3
- JavaScript (ES6)
- Shopify AJAX Cart API
- Shopify Theme Architecture
